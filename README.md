# 明德学院 · 社团与活动报名管理平台

> 软件工程期末项目 — 纯前端原型（HTML5 + CSS3 + 原生 JavaScript，零框架依赖）

---

## 项目简介

本系统为明德学院的社团与活动报名管理平台，分为**用户端**（浏览活动、报名、查看记录）和**管理端**（审核报名、发布活动、成员管理），共 6 个功能页面 + 1 个统一展示框架页，合计 7 个 HTML 文件。

### 技术栈

| 技术 | 说明 |
|------|------|
| **HTML5** | 语义化标签，iframe 嵌入式框架 |
| **CSS3** | Flexbox + Grid 布局，响应式适配桌面/平板/移动端 |
| **原生 JavaScript** | 无任何前端框架或 CSS 框架 |
| **主色调** | `#003366` 藏青色，学术风格 |

## master.html — 统一展示框架（答辩入口）

**这是本次新增的页面，为期末答辩设计的统一展示入口。** 通过 iframe 画中画技术将所有子页面嵌入同一窗口，左侧配有统一导航菜单。

### 架构设计

```
┌────────────────┬──────────────────────────────┐
│                │                              │
│   左侧导航栏    │       iframe 内容区          │
│   (240px)      │       (自适应)               │
│                │                              │
│  · 登录页面     │   加载 login.html            │
│  · 系统首页     │   加载 index.html            │
│  · 活动详情     │   加载 detail.html           │
│  · 个人中心     │   加载 profile.html          │
│  · 发布新活动   │   加载 create-activity.html  │
│  · 管理仪表盘   │   加载 admin-dashboard.html  │
│                │                              │
└────────────────┴──────────────────────────────┘
```

### 核心功能

- **左侧导航栏**：深色主题（`#001f3f`），6 个菜单项，每个标注了对应成员姓名
- **iframe 嵌入**：默认加载 [login.html](login.html)，所有子页面在 iframe 内完整运行
- **菜单高亮同步**：
  - 点击左侧菜单 → 切换 iframe 页面 + 高亮对应菜单
  - iframe 内部发生页面跳转（如登录成功后跳转首页）→ 自动检测并更新左侧菜单高亮
  - 通过读取 `iframe.contentWindow.location.href` 提取当前文件名实现
- **安全策略说明**：使用 `file://` 协议直接打开时，浏览器跨域限制会阻止读取 iframe URL，建议通过 Live Server 运行

### 样式特点

- 左侧导航栏使用比子页面更深的 `#001f3f` 背景，形成层次对比
- 菜单项悬停有半透明高亮，激活项使用主题色 `#003366` + 左侧蓝色边框 `#4da6ff`
- 全窗口无溢出滚动条（`overflow: hidden`），由 iframe 内部自行滚动

## 各功能页面详解

### 1. login.html — 登录/注册

**作者：张涵博**

| 功能 | 说明 |
|------|------|
| 角色切换 | 普通用户 / 管理员（藏青色按钮组，带阴影过渡动画） |
| 登录/注册 Tab | 双面板切换，激活 Tab 底部有蓝色下划线指示器 |
| 登录表单 | 用户名 + 密码，空值校验 + Toast 提示 |
| 注册表单 | 用户名 + 学号 + 密码 + 确认密码 |
| 表单验证 | 空值检查、学号纯数字格式校验、密码≥6位、两次密码一致性校验 |
| 登录跳转 | 普通用户 → [index.html](index.html) / 管理员 → [admin-dashboard.html](admin-dashboard.html) |
| 回车提交 | 所有输入框支持 Enter 键直接提交 |

**样式**：渐变蓝色背景 + 居中白色卡片 + 圆角阴影

### 2. index.html — 首页

**作者：陈清博**

| 功能 | 说明 |
|------|------|
| 顶部导航栏 | sticky 固定，Logo + 首页/我的社团/个人中心链接 + 用户头像 + 退出按钮 |
| Hero 区域 | 深蓝渐变背景 + 活动搜索框（橙色搜索按钮 `#e67e22`） |
| 分类筛选 | 6 个筛选标签：全部 / 学术 / 体育 / 艺术 / 科技 / 志愿 |
| 活动卡片网格 | CSS Grid 自适应列（`auto-fill, minmax(280px, 1fr)`），8 条模拟数据 |
| 卡片内容 | 配色封面 + 分类标签 + 报名状态徽章 + 标题 + 日期时间 + 主办社团 + 剩余名额 |
| 搜索功能 | 实时过滤（`input` 事件），按活动标题/社团/关键词匹配卡片文本 |
| 移动端适配 | ≤768px 隐藏导航链接，显示汉堡菜单 → 右侧抽屉式导航 |
| 页脚 | 深蓝背景版权信息 |

**模拟数据**：8 条活动覆盖全部 5 个分类，其中 1 条（AI讲座）标记为已满员

### 3. detail.html — 活动详情与报名

**作者：张涵博**

| 功能 | 说明 |
|------|------|
| 全宽渐变海报 | 根据活动 ID 动态切换渐变色（每项活动有独立的 `color1`/`color2`） |
| URL 参数加载 | 通过 `?id=` 参数匹配活动数据（1-8），无匹配时默认显示 ID=1 |
| 活动详情区 | 标题 + 描述文本 + 活动内容列表 + 注意事项 |
| 右侧信息卡 | 日期 / 时间 / 地点 / 主办社团 / 剩余名额（橙色大号数字） |
| 报名表单 | 姓名 + 学号 + 联系方式 + 备注（选填），含表单验证 |
| 满员处理 | `remain === 0` 时隐藏报名表单，显示"报名已截止"提示 |
| 报名提交 | 成功后 Toast 提示 + 清空表单 |

**数据规模**：8 条活动完整数据，含标题、日期、时间、地点、社团、名额、状态、描述、渐变色

### 4. profile.html — 个人中心

## 页面跳转关系

```
                     master.html (统一框架入口)
                           │
                           │ iframe 嵌入所有子页面
                           ▼
                     login.html
                    /          \
           [普通用户]         [管理员]
              │                  │
              ▼                  ▼
         index.html      admin-dashboard.html
         /      \           /        │        \
        ▼        ▼         ▼         ▼         ▼
  detail.html  profile  dashboard  create-  members
  ?id=X        .html   (审核表格)  activity  (成员表)
                            .html
```

**iframe 内跳转自动同步**：当子页面在 iframe 内发生跳转（如 login → index），master.html 的左侧菜单会自动跟随高亮。

## 响应式设计

| 断点 | 适配行为 |
|------|----------|
| **桌面端（>768px）** | 完整多列布局，侧边栏固定，活动卡片网格多列 |
| **移动端（≤768px）** | 所有页面统一采用以下策略 |

- **导航栏** → 汉堡菜单 + 抽屉式侧边栏 / 滑出式导航
- **活动卡片** → 单列排列
- **详情页** → 侧边栏移至内容下方（单列布局）
- **个人中心** → 信息卡垂直排列居中
- **管理端** → 侧边栏 `translateX(-100%)` 隐藏，按钮滑出 + 遮罩层
- **表单** → 双列变单列（`grid-template-columns: 1fr`）
- **操作按钮** → 纵向排列

1. **推荐**：使用 VS Code Live Server 插件打开 [master.html](master.html)（可正常使用 iframe URL 检测功能）
